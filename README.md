## spring-security
spring-security 5.2.1 버전 학습

## 환경
- Spring-boot 2.2.1.RELEASE
- Spring-security 5.2.1.RELEASE
- JDK 1.8

## 학습목표
1. 스프링 시큐리티 5.x 버전에 전반적 이해
2. 인증 및 인가 처리 프로세스 이해
3. 시큐리티 적용한 세션 방식 프로젝트 적용
4. 5.x 버전 학습 후 스프링 부트 3버전 및 시큐리티 6.x 버전 마이그레이션
